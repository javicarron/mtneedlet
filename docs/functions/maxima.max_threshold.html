maxima.max_threshold¶

max_threshold(maxima, threshold)[source]¶

Get the maxima with intensity above or equal to a threshold

Parameters:
  • maxima (pd.DataFrame) – Pandas DataFrame with the information of the maxima. Has to contain at least the column ‘Intensity’.
  • threshold (float) – Minimum value of the maxima to be returned.
Returns:

The same Pandas DataFrame as the input maxima, but containing only the maxima with an intensity equal or greater than threshold.
Return type:

pd.DataFrame
