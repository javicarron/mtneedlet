Extra tools¶

One extra function is provided with MTNeedlet. This functions creates a visual and easy way to analyse a particular maximum. It can also be used as an easy tool to inspect anomalies.

Visualization¶

maximum_info(maxima, n, mapp[, nmax])

Show a plot with a maximum and some information about it.
